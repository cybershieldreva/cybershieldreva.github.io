How can I be aware of Pharming.?


Pharming is a cyberattack intended to redirect a website's traffic to another, fake site .Pharming attacks involve redirecting user requests by manipulating the DNS protocol and rerouting the target to its intending IP address Pharming attacks involve redirecting user requests by manipulating the DNS protocol and rerouting the target to its intending IP address. Pharming can be conducted either by changing the hosts file on a victim's computer or by exploitation of a vulnerability in DNS server software.
